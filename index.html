Why does it work though?

  • Processor state can be corrupted
  • But how does voltage glitches cause this?
  • We need to look at transistors
  • ... and logic gates

NAND Logic

C = !(A & B)

A B C
0 0 1

1 1 0

A = 0, B = 0: Switching

A = 0, B = 0: Stable

A = 1, B = 1: Switching

A = 1, B = 1: Stable

A = 1, B = 0: Switching

A = 1, B = 0: Stable

A = 1, B = 0: Glitch Failed

A = 1, B = 1: Stable

A = 1, B = 0: Glitch Successful

Takeaway

  • Voltage glitching (mostly) only affects gate outputs...
  • that are switching ...
  • from 0 to 1 ...
  • at the precise moment your glitch is injected
  • Good news: most system state isn't corrupted

Timing is important

  • Control when and for how long the glitch occurs
  • Too soon: no effect
  • Too short: no effect
  • Too long: system shuts down
